## Handle Command

You can provide your handler by using `commandHandler` [prop](#props).
Your function should returns `true` if the command is manually handled. Otherwise, commands will be executed by default handler.

`commandHandler` prop also has a built-in `println` function (2nd parameter), you can call it to print something to the terminal.

### Default Commands

- `clear` - clear the terminal
- Others - println: "command not found".
