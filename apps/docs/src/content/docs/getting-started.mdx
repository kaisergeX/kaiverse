`@kaiverse/k` is a collection of powerful utilities, fully featured React components, and hooks.

## Installation

<Tabs syncKey="pkg">
<TabItem label="pnpm">

```sh
pnpm add @kaiverse/k
```

</TabItem>

<TabItem label="npm">

```sh
npm i @kaiverse/k
```

</TabItem>

<TabItem label="Yarn">

```sh
yarn add @kaiverse/k
```

</TabItem>
</Tabs>

## Packages

- `@kaiverse/k/ui`: React components.
- `@kaiverse/k/hooks`: Hooks.
- `@kaiverse/k/utils`: Utility functions.
